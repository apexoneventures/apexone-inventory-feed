# ApexOne Inventory Feed API

A production-ready inventory management system for automotive dealerships.

## Overview

ApexOne Inventory Feed API enables dealerships to sync live vehicle inventory from their DMS and serve it to showroom tools, web applications, and mobile apps via a simple REST API.

**Status:** Production Ready  
**Version:** 1.0.0

## Quick Start

### Deploy

```bash
git clone https://github.com/apexoneventures/apexone-inventory-feed.git
npm install
npm start
```

Runs on `http://localhost:3000`
